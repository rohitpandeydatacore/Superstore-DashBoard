## **Super Store Sales Dashboard**



#### **Project Overview:-**



###### This is a beginner-level Excel dashboard project created using the SuperStore Sales dataset.

###### The dashboard helps in analyzing sales, profit, regions, and product categories using charts and Pivot Tables.



#### **Objective:-**



###### To learn data analysis and dashboard creation in Microsoft Excel.



#### **Tools Used:-**

###### \-Microsoft Excel

###### \-Pivot Tables

###### \-Pivot Charts

###### \-Slicers

###### \-Conditional Formatting



#### **Dashboard Features**

###### \-Sales Analysis

###### \-Profit Analysis

###### \-Regional Analysis

###### \-Category-wise Sales

###### \-Monthly Sales Trends

###### \-Interactive Slicers



#### **Key Insights**

###### \-Technology category had the highest sales.

###### \-Furniture category had lower profit.

###### \-West region generated the highest revenue.

###### SuperStore Sales Dataset
